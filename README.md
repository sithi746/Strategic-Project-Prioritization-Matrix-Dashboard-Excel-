# Strategic-Project-Prioritization-Matrix-Dashboard-Excel
## Excel dashboard designed to evaluate and prioritize projects based on Business Impact, Feasibility, and Project Size, enabling organizations to make informed strategic decisions.

### Project Description:

Organizations often have multiple initiatives competing for limited resources. Choosing which projects to execute first can be challenging without a structured decision-making framework.

This dashboard provides a Project Prioritization Matrix that visually categorizes projects into strategic quadrants based on Impact and Feasibility, while using bubble size to represent Project Size. It helps decision-makers quickly identify high-value opportunities, major investments, and lower-priority initiatives.

### 🛠 Tech Stack:

This dashboard was built using the following tools and technologies:

📊 Microsoft Excel 365 – Dashboard development 
📈 Bubble Charts – Visual representation of Impact, Feasibility, and Project Size
🎨 Conditional Formatting – Data bars and color-based visual cues
📋 Excel Tables – Structured data management
🔍 Chart Formatting & Custom Labels – Enhanced readability and presentation
📐 Dashboard Design Principles – Clean, business-focused layout


### Source: 
Self-created sample business dataset for portfolio purposes.
The dataset contains project-level information.

### Features / Highlights:
📌 Business Problem

Organizations frequently manage numerous projects while operating under limited budgets, time, and resources.

Without a structured prioritization framework, it becomes difficult to determine:

Which projects should be executed first?
Which initiatives provide the highest business value?
Which projects require significant investment?
Which projects can be postponed?

🎯 Goal of the Dashboard

The dashboard helps users:

Prioritize projects objectively.
Compare Impact and Feasibility simultaneously.
Visualize project size through bubble dimensions.
Identify Quick Wins and Major Projects.
Support strategic planning and resource allocation.
 
📊 Dashboard Walkthrough
📈 Strategic Project Priority Matrix

The main Bubble Chart categorizes projects into four strategic quadrants:

🟢 Quick Wins
High Impact
High Feasibility

🔵 Major Projects
High Impact
Lower Feasibility

🟡 Nice to Have
Lower Impact
High Feasibility

🔴 Low Priority
Lower Impact
Lower Feasibility
Bubble Size represents the overall Project Size.
